# 🧑🏻‍🌾 팜인포(Jeju)

<img width="1269" alt="farminfo" src="https://github.com/user-attachments/assets/a911e2eb-397f-4079-bec9-70267f6d3dae">

`팜인포(Jeju)`는 제주도 농업 데이터를 디지털화하여 재배 면적 신고와 출하량 신고 과정을 간소화하고, 농민들에게 실시간으로 적정 재배 면적과 예상 가격 정보를 제공합니다. 이를 통해 농업 생산성과 수익성을 높이고, 불필요한 인력과 비용을 절감하며, 농작물 가격 안정화와 농가의 부채 문제 해결에 기여하는 스마트 농업 플랫폼입니다.

## 🥬 서비스 및 기능 소개

팜인포(Jeju)는 농민 사용자와 관리자(지자체 소속) 사용자로 나누어 서비스를 활용할 수 있도록 했습니다.
